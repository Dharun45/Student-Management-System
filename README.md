# Student-Management-System
Developed a console-driven application using Core Java to efficiently manage student information. Implemented CRUD functionalities, automated marks and grade computation, and utilized ArrayList with OOP principles for structured data handling. Designed the system with a modular architecture and a clean, user-friendly text-based interface.

*The Key motive of this project*
The primary objective of the Student Management System project is to simplify and automate the process of handling student information. Traditionally, managing student records manually can lead to data inconsistency and time-consuming operations. This project aims to provide a structured, digital solution that allows users to easily add, update, view, and delete student details efficiently.
